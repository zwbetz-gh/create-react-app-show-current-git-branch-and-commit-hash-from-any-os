# Create React App: Show Current Git Branch and Commit Hash From Any OS

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Sister Links

## Run it

1. Install NodeJS version 12 or higher
1. Run `npm install`
1. Run `npm start`

## Relevant Files

- `package.json`
- `src/gitInfo.js`
- `src/App.js`
